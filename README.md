##  总体设计
流水线是数字系统中一种提高系统稳定性和工作速度的方法，广泛应用于高档CPU的架构中。  
根据MIPS处理器的特点，将整体的处理过程分为取指令（IF）、指令译码（ID）、执行（EX）、存储器访问（MEM）和寄存器写回（WB）五级，对应多周期的五个处理阶段。  
其中，一个指令的执行需要 5 个时钟周期。
